# example-web-app 概览

> 项目入口文档。第一次接触项目或解释架构时读这里。

## 一句话定位

订单系统的用户端 Web 应用，浏览订单、下单、支付、查看物流。

## 代码位置

- **仓库路径**：`~/code/example-web-app`
- **主语言**：TypeScript + React 18
- **主要依赖**：Vite、TanStack Query、grpc-web、[[项目/example-shared-proto/概览]]

## 架构速览

SPA，路由用 React Router v6。数据层 TanStack Query 管缓存，gRPC-Web 调 [[项目/example-api-server/概览]]。proto 类型由 example-shared-proto 自动生成。

## 关键模块

| 模块 | 路径 | 职责 |
|------|------|------|
| 路由 | `src/routes/` | 页面入口 |
| 数据层 | `src/api/` | TanStack Query hooks 封装 gRPC 调用 |
| 组件库 | `src/components/` | 通用 UI |
| 业务模块 | `src/features/order/` | 订单相关页面和逻辑 |

## 跨仓库依赖

- [[项目/example-shared-proto/概览]] — TypeScript 类型由 proto 自动生成
- 调用 [[项目/example-api-server/概览]] 的 gRPC 接口
