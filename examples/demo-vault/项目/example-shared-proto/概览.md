# example-shared-proto 概览

> 项目入口文档。第一次接触项目或解释架构时读这里。

## 一句话定位

订单系统所有服务共用的 gRPC 接口定义，提供 Go 和 TypeScript 双语言代码生成。

## 代码位置

- **仓库路径**：`~/code/example-shared-proto`
- **主语言**：Protocol Buffers 3
- **构建工具**：buf + protoc-gen-go + protoc-gen-ts

## 架构速览

按业务域分目录：`order/`、`payment/`、`user/`。每个域一个 `.proto`。CI 自动生成 Go 包发布到内部 module proxy，TS 包发到 npm registry。

## 关键模块

| 模块 | 路径 | 职责 |
|------|------|------|
| 订单接口 | `order/v1/order.proto` | 订单 CRUD、状态枚举 |
| 支付接口 | `payment/v1/payment.proto` | 支付单据、回调 |
| 用户接口 | `user/v1/user.proto` | 用户信息 |
| 通用类型 | `common/v1/types.proto` | 分页、错误码 |
